#Separator 
A Settings Separator Field

## Parameters
Parameter | Type | Value
--- | --- | ---
type | `required` | Predefined String (textarea)
title | `optional` | String
desc | `optional` | String

```php
'addon_separator' => array(
    'type' => 'separator',
    'title' => __('Separator Field','your-textdomain'),
)
```
